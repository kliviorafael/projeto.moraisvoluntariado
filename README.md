# projeto.moraisvoluntariado

Projeto para avaliação final da disciplina de Linguagem em Programação 2 - UNIESP

Alunos: Altayr, Francisco, Klivio

Foi recebida a proposta de construir uma aplicação para uma Ong, a qual vinhesse a facilitar o cotidiano da entidade não governamental. Como tecnologia utilizada, a linguagem de programação Java foi escolhida, pois foi trabalhada pela disciplina e tinhamos que aplicar todo o conhecimento adquirido no período em relação a construção de sistemas, sob o paradigma da Orientação a Objeto.

Em específico nesse projeto, utilizamos o Framework ItextPDF para a criação de arquivo .PDF do relatório financeiro.
